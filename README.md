## 这套系列的逻辑

```
① 用工具      → Claude Code 是个 harness,你先学会用它
② 用接口      → 直接调模型能力,手写 agent 循环
③ 扩能力      → 用 MCP 把外部系统接进生态(写一次处处可用)
④ 会沟通      → prompt engineering 贯穿所有场景
⑤ 搭系统      → 把以上组装成可靠的 agent 架构
⑥ 带团队      → 把 Claude Code 落地进真实(混合工具)团队的日常开发
⑦ 落领域      → 落到前端:工程实践 + 构建选型 + 规格驱动(让 AI 不跑偏)
```

第 ⑤ 册第 8 章末尾有一张"完整 AI 产品全景图",把五册串起来——如果你在构建 AI 产品,可以从那里反查每个环节对应哪册。

## 怎么用

- **零基础新手**:先走上面 🔰 的 [新手上路](07-claude-前端开发-手册/示例项目/00-新手上路-从零搭建.md),把环境和第一个项目跑通,再回来按顺序深入。
- **有基础**:从 ① 开始按顺序读,每册 README 里有 3–4 周的学习路径和练习。
- **找特定主题**:每册都有「附录-速查」,是查命令/参数/模式的工具书。
- **要跑代码**:涉及完整可运行程序的章节,代码放在该手册的 `示例代码/` / `示例项目/` / `模板/` 文件夹里,可直接运行。

## 规模

七册共约 60 章、14000+ 行,基于 Claude Code v2.1.x、Anthropic 官方 API/SDK 文档、MCP 官方文档、Anthropic 工程文章、及前端生态(React 19/Vite/Next)与 OpenSpec 的当前现状编写(2026-06)。涉及模型 ID、API 参数、SDK 用法、版本现状的部分均核对过官方文档/实时搜索。
